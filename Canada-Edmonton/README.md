# エドモントン旅行しおり

森下枝里子さんのカナダ・エドモントン旅行しおり（2025年3月3日〜8日）

## 概要

夫・恭祐から妻・枝里子への旅行ガイド。高校時代のホームステイ以来、約20年ぶりのカナダ再訪。エドモントン在住のイチローさん家族との再会がメインイベント。
